# Les Douze Coup d'YNOV
Ce jeu reprend le concept de l'émission télévisée des 12 Coups de midi sur TF1.


# Installation
Ce jeu a besoin de 2 répositories, vous devez les cloner dans le même dossier Frontend : 

```
git clone https://github.com/LesDouzeCoupsd-YNOV/Frontend
```
```
git clone https://github.com/LesDouzeCoupsd-YNOV/Backend
```

# Lancement Frontend
Aller dans le dossier Frontend et rentrer cette commande dans le terminal : 
live-server --open=index.html

# Lancement Frontend
Aller dans le dossier Backend et rentrer cette commande dans le terminal :
npm start

# Règles
Les règles sont simples, vous devez répondre à des questions de culture générale. Mais ce n'est pas tout ! Sur le côté gauche , vous avez une image cachée et une liste de personnalités. Plus vous allez répondre bon aux questions, plus l'image sera visible et vous pourrez déterminer la personnalité. Si vous trouvez vite la personnalité, vous gagnerez plus de points
